# AJIS

AJIS is a lightweight, structured text format with an optional binary container layer.

## Repositories using AJIS
- AS_426 (DbOS runtime)
- BOS
- AfNAS-OS
- Tools (validators, converters, benchmarks)

## Structure
- `c/` – C implementation (lexer/parser/serializer)
- `tests/` – conformance test data
- `docs/` – format documentation

## Status
Work in progress. Text core is stable-ish; binary container design is being finalized.
